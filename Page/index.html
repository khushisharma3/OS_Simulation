Page Replacement

Page replacement is a process of swapping out an existing page from the frame of a main memory and replacing it with a new page. It is required when all the frames of main memory (or the ones alloted for the process) are already occupied. Thus, a page has to be replaced to create a room for the required page.

SIMULATE

FIFO

First In First Out

In this algorithm,the OS keeps track of all pages in the memory in a queue, with the oldest page in the front of the queue. When a page needs to be replaced, the page at the front of the queue is selected for removal.

LRU

Least Recently Used

This algorithm replaces the page which has not been referred for the longest time.

Optimal

Optimal

This algorithm replaces the page which will not be referred for the longest in future. It exists only in theory and is used as a benchmark
