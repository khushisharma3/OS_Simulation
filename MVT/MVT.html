MVT

Multiprogramming with a Variable number of Tasks

In the MVT scheme, the OS keeps track of which parts of memory are available and which are occupied. Initially, all memory is available for user processes and is considered one large block of available memory, a hole. Eventually, as the processes start entering and leaving the memory,there will be a set of holes of various sizes.
